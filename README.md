# 系统架构设计师考试备考助手

基于 MCP 的智能备考辅导工具，采用苏格拉底教学法帮助考生高效备考软考高级——系统架构设计师。

## 功能特点

- **引导式学习**：通过问答引导考生主动思考，而非被动接收信息
- **知识域全覆盖**：涵盖7大知识域，包含详细的考点清单
- **案例分析训练**：提供四层答题框架和专项练习
- **论文写作指导**：结构化模板和写作要点分析
- **个性化规划**：根据考生情况制定学习计划

## 适用人群

- 首次备考系统架构设计师的考生
- 需要强化案例分析和论文写作的考生
- 希望通过引导式学习巩固知识点的考生

## 快速开始

### 使用方式

本项目为 MCP Skill 文件，需配合支持 MCP 协议的 AI 助手使用。

将 `SKILL.md` 内容配置为 AI 助手的 system prompt 即可开始使用。

### 学习路径建议

1. **基础阶段**（4周）：通读教材，建立知识框架
2. **突破阶段**（4周）：按权重攻克重点知识域
3. **强化阶段**（3周）：案例分析和论文专项训练
4. **冲刺阶段**（1周）：错题复习和心态调整

## 知识域权重

| 优先级 | 知识域 | 权重 |
|--------|--------|------|
| 最高 | 系统架构设计 | 25% |
| 高 | 软件架构设计 | 20% |
| 中 | 需求工程与建模 | 15% |
| 中 | 系统设计 | 12% |
| 中 | 架构评估与测试 | 10% |
| 中 | 云计算/大数据/AI | 8% |
| 低 | 信息安全架构 | 7% |
| 低 | 项目与团队管理 | 3% |

## 考试概况

| 科目 | 满分 | 及格线 | 题型 |
|------|------|--------|------|
| 综合知识（上午卷） | 75分 | 45分 | 选择题（75道） |
| 案例分析（下午I） | 75分 | 45分 | 问答题（5选3） |
| 论文（下午II） | 100分 | 45分 | 论文写作（4选1） |

## 核心内容

- **知识域详解**：7大知识域的详细考点清单和验证问题
- **案例分析**：四层答题结构、常见案例类型、答题技巧
- **论文写作**：结构模板、常见方向、写作要点
- **学习资源**：官方教材、历年真题、辅助资料推荐
- **资料管理**：支持关联用户本地备考资料，辅助讲解
- **进度追踪**：自动记录学习历史，支持上下文回顾
